데이터 연동/연동컴포넌트 매핑 가이드

앱과 앱을 연동할 수 있는 컴포넌트로
다른 앱의 데이터를 쉽게 참조할 수 있어요!

  • 데이터 연동

    앱의 입력화면에 '데이터 연동' 컴포넌트를 추가하면 다른 앱과 연동되면서 연동된 앱의 데이터를 검색할 수 있는 검색 버튼이 추가됩니다.
    데이터 연동이 안 된다면 연동 허용 권한을 확인하세요.자세히 보기

  • 연동컴포넌트 매핑

    '데이터 연동'을 한 후에, '연동컴포넌트 매핑' 컴포넌트를 추가하면 연동된 앱의 특정 데이터 필드 값을 추가로 더 가져올 수 있습니다.
    '데이터 연동' 컴포넌트가 추가되지 않은 상태에서는 동작하지 않습니다.자세히 보기
