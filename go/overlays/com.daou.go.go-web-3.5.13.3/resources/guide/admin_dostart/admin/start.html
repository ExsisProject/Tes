  1. 계정/조직도 설정

    다우오피스 로그인을 위한 멤버(계정) 및 부서, 용량 설정

  2. 근태/연차 설정

    임직원들의 근무시간 및 초기 연차 설정

  3. 메일 도메인 연결

    다우오피스에서 신청한 도메인으로 메일을 수/발신

  4. 다우오피스 오픈

    전임직원에게 그룹웨어 오픈 공지 등
