다우오피스 시작하기

메일 도메인 연결

  • 1. MX레코드 설정
  • 2. SPF레코드 설정
  • 3. 확인하기

다우오피스에서 메일을 수/발신 하려면?

도메인 구매업체에서 MX레코드 및 SPF레코드를 변경(등록)해 주셔야 합니다.

  1. MX레코드 설정

    외부에서 이메일을 수신받기 위한 필수 설정 사항

  2. SPF레코드 설정

    외부로 메일을 전송하기 위한 권장 설정 사항
네임서버의 MX 레코드 변경(등록)은 반드시 다우오피스의 계정을 먼저 등록한 후, '이메일을 받고자 하는 시점'에 설정을 변경(등록)해 주시기 바랍니다.
※ 다우오피스에서 외부 메일 수/발신을 하지 않는다면, MX레코드 및 SPF레코드는 설정하지 않으셔도 무방합니다.
