3. 확인하기

3 - 3
MX레코드 및 SPF레코드가 정상적으로 설정되었는지
DNS 조회를 통해서 확인할 수 있습니다.
※네임서버(DNS)의 정보 변경 및 적용은
  최소 1시간, 최대 72시간(3일)까지 소요될 수 있습니다.
[MX레코드 반영 확인]
1) 윈도우 실행(R)창에서 “cmd” 입력후 엔터
2) 명령 프롬프트창에서 ”nslookup -type=mx 도메인명” 입력
3) 레코드값 확인
[SPF레코드 반영 확인]
1) 윈도우 실행(R)창에서 “cmd” 입력후 엔터
2) 명령 프롬프트창에서 ”nslookup -type=txt 도메인명” 입력
3) 레코드값 확인
