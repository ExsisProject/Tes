계정 조직도 설정

2. 조직설계

2 - 5
부서별로 임직원을 관리하기 위해 필요한 작업입니다.
부서 및 부서원을 추가할 수 있습니다.

용어

  • 부서메일아이디 : 부서원에게 메일 발송을 위한 주소
  • 부서코드 : 부서 일괄등록 시 구분하기위한 키 값
  • 부서약어 : 부서명대신 전자결재 문서번호에서 사용
[조직관리 > 조직설계]
조직도 목록의 [+]버튼을 눌러 부서명을 입력 후 저장합니다.
[조직관리 > 조직설계]
추가한 부서를 선택 후 [+신규 부서원 생성] 버튼을 클릭하여
부서원을 추가합니다.
[조직관리 > 조직설계]
필수입력 항목들과 추가한 직위/직책을 설정합니다
※ 메일그룹,부가정보 등은 '상세 정보' 클릭 후 설정
※'멤버통합관리' 메뉴에서도 추가 가능
