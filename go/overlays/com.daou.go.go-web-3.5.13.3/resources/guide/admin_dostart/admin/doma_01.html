1 - 3
고객사의 다우오피스 메일 계정으로 메일을 수신하려면
'MX레코드 등록'이 필수입니다.
네임서버에 다우오피스의 MX레코드만 설정해주세요.

용어

  • MX레코드 : ‘메일이 수신될 목적지’를 결정하는 정보
